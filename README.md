# Triotec Engenharia - site (one-page)

Este repositório contém o site one-page para Triotec Engenharia (deploy gratuito via Netlify).

Arquivos principais:
- index.html
- triotec-colors.css
- triotec-logo.svg
- triotec-favicon.svg
- admin/ (Netlify CMS)

Deploy no Netlify (resumo):
1. Acesse https://app.netlify.com/new
2. Escolha GitHub e autorize se necessário
3. Selecione o repositório "randalfilho1981-rgb/triotec-engenharia-site"
4. Deploy automático será criado. Padrão: publicar a raiz ("publish = .").

Configurar Netlify Forms notificações:
- No painel do site no Netlify, vá em Forms -> Settings -> Notifications e adicione o e-mail eng.triotec@gmail.com para receber notificações de formulários.

Configurar Netlify CMS (opcional):
- Ative Netlify Identity nas configurações do site e em seguida habilite Git Gateway para permitir edição via CMS.
